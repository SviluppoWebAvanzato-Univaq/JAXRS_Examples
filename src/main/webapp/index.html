Jax-RS Test Server

JAX-RS template application

This is a template web application based on JakartaEE 9 that runs on Apache Tomcat 10 with Java 14 or above.

The application is configured to expose RESTful resources through the JAX-RS reference implementation, Jersey (version 3, for Jakarta EE).

JAX-RS JSON serialization is handled by the Jackson JSON Provider (version 2, for Jakarta EE).

You can read the automatically-generated OpenAPI documentation at the address /rest/openapi.json
